## ssm  个人记账管理系统

## 一、系统介绍

用户：记账、收支管理、收支统计、收支分析、收支预算、心愿单、备忘录

管理员：用户管理、收支类别管理、新闻管理

## 二、所用技术

后端使用 ssm(spring+springmvc+mybatis)，前端使用 bootstrap 组件库配合完成开发。

## 三、环境介绍

基础环境 :IDEA/eclipse，maven3.x，JDK 1.8 ， Mysql 5.7

源码+数据库脚本
